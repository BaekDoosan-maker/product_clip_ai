# 플로팅 버튼 
— 유사 이미지 검색 기능 적용

설명

- 화면의 상품 카드에 표시된 플로팅(돋보기) 버튼을 누르면 해당 상품 이미지를 서버로 업로드해 유사 이미지 검색을 수행하고, 결과를 모달로 보여줍니다.
<img width="1912" height="903" alt="image" src="https://github.com/user-attachments/assets/e6e32f60-f356-4814-a3f3-08a372d78ac8" />

- 모달에는 유사 이미지들의 썸네일과(가능하면) 매칭된 상품 정보(name, price)를 표시합니다. 
<img width="1916" height="909" alt="image" src="https://github.com/user-attachments/assets/9ef8d61e-d451-496f-8908-0d34f4cdce42" />

- 상품 정보를 클릭하면 상품 상세 페이지로 이동합니다.
